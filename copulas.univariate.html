copulas.univariate.kde module¶

class copulas.univariate.kde.KDEUnivariate[source]¶

Bases: copulas.univariate.base.Univariate

A wrapper for gaussian Kernel density estimation implemented in scipy.stats toolbox. gaussian_kde is slower than statsmodels but allows more flexibility.

fit(column)[source]¶

Fit Kernel density estimation to an list of values.

:param : param column: list of datapoints to be estimated from. :param : type column: 1-D np.ndarray or pd.Series or list

This function will fit a gaussian_kde model to a list of datapoints and store it as a class attribute.
