class copulas.bivariate.clayton.Clayton(copula_type=None)[source]¶

Bases: copulas.bivariate.base.Bivariate

Class for clayton copula model.

compute_theta()[source]¶

Compute theta parameter using Kendall’s tau.

On Clayton copula this is \(τ = θ/(θ + 2) \implies θ = 2τ/(1-τ)\) with \(θ ∈ (0, ∞)\).

On the corner case of \(τ = 1\), a big enough number is returned instead of infinity.

compute_theta()[source]¶

Compute theta parameter using Kendall’s tau.

On Frank copula, this is \(τ = 1 − \frac{4}{θ} + \frac{4}{θ^2}\int_0^θ \! \frac{t}{e^t -1} \, \mathrm{d}t.\).
